### Attributes

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| -- | -- | -- | -- | -- |
| data-source | 列表数据源 | Array | - | [] |
| rowKey | 各项 key 的取值 | String \| Function | - | - |
| split | 是否展示分割线 | Boolean | - | true |
| max-height | List 的最大高度 | String \| Number | - | - |
| pagination | 对应 BasePagination 配置，设置 false 不显示 | Boolean \| Object | - | [] |
