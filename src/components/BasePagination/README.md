### Attributes

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| -- | -- | -- | -- | -- |
| showPageMsg | 是否显示分页信息 | Boolean | - | false |

### Events

| 事件名 | 说明 | 参数 |
| -- | -- | -- |
| page-change | 修改当前页 | - |
| size-change | 修改每页条数 | - |
| prev-click | 上一页 | - |
| next-click | 下一页 | - |

具体内容参考 [Pagination](https://element.eleme.cn/#/zh-CN/component/pagination)
